Spotify Listening Analytics Dashboard
Project Overview -

This project is an interactive Spotify Listening Analytics Dashboard built using Power BI. The dashboard analyzes listening behavior, streaming trends, user engagement, and platform usage through interactive visualizations and KPI tracking.

Tools Used -
Power BI
SQL
Data Visualization

Key KPIs - 
Total Streams
Unique Tracks
Unique Artists
Total Listening Hours

Dashboard Features -
Monthly listening analysis
Yearly trend filtering
Skip analysis
Top streamed artists
Platform-wise listening distribution
Hourly listening patterns

Key Insights -
Peak listening activity occurs during evening hours
Android users contribute the highest streaming activity
Skip rate is significantly lower than completed streams
Top artists generate the majority of streams
